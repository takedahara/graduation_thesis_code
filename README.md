卒業研究「プライバシー保護に考慮したカメラ画像を用いた機械学習による被介護者の異常検知」において、作成したコードをまとめています。
label_creation.pyは異常の種類を表すラベルを作成するためのプログラムです。
rnn.pyはlabel_creation.pyによって、作成されたラベルを用いて、異常検知を行うための機械学習を実行するプログラムです。入力データは、動画のフレーム間差分をとったものそれぞれについて、ピクセルの和を取ることで１次元データに変換したものです。１次元の時系列データをRNNモデルを用いて、異常を分類しています。
simplenn.pyはlabel_creation.pyによって、作成されたラベルを用いて、異常検知を行うための機械学習を実行するプログラムです。入力データは、動画のフレーム間差分をとったものそれぞれについて、縦のピクセルの和を取ることで多次元データ（１９２０次元）に変換したものです。多次元データをシンプルなニューラルネットワークモデルを用いて、異常を分類しています。
